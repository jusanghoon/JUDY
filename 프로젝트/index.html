시작 지점

이 파일은 새로운 웹 프로젝트의 기본 HTML 구조입니다. Tailwind CSS가 연결되어 있어 바로 다양한 스타일을 적용할 수 있습니다.

다음 단계:

  • 필요한 HTML 요소들을 `main` 태그 안에 추가하세요.
  • JavaScript 기능을 구현하고 싶다면, 아래 `script` 태그를 사용하거나 별도의 `.js` 파일을 연결하세요.
  • 더 복잡한 스타일이 필요하다면, 별도의 `style.css` 파일을 만들고 `` 태그 안에서 연결할 수 있습니다.
